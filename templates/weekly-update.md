# Weekly update template

A shorter sibling of `biweekly-update.md`. Friday-morning send. Three paragraphs max.

Subject line:

```
Rockcrete USA Website Rebuild weekly: {{ONE_PHRASE_HEADLINE}}
```

Body:

```
Hi {{CLIENT_PRIMARY_CONTACT}},

{{HEADLINE_SENTENCE}}

This week:
- {{SHIPPED_1}}
- {{SHIPPED_2}}
- {{SHIPPED_3}}

Next week:
- {{NEXT_1}}
- {{NEXT_2}}

{{OPTIONAL_CONCERN_OR_ASK_SENTENCE}}

Live dashboard: https://rockcreteusa.projectizer.ai/

Best,
Daniel Gonell
```

That's it. No tables, no traffic lights, no metrics. The dashboard carries the detail; this email's job is to keep the relationship warm and surface anything blocking.

## When to upgrade to the biweekly format

If any of these are true on a Friday:

- A milestone shipped or slipped.
- Status moved between green / yellow / red.
- A payment milestone became due.
- A decision is needed from the client.

... skip the weekly format and send the biweekly format instead.

## After sending

Same as biweekly: log in `stakeholder-register.md` "Communications log", commit and push.
